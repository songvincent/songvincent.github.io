# 迭代器

** 定义：** 迭代器 iterator 是面向对象的程序设计语言都提供的遍历序列对象的一种方法，在 Python 中封装程度更高，其把迭代器协议在语言的层面就已经实现了，所以使用起来比其他语言方便的多。

  • 迭代器是访问可迭代对象的工具
  • 迭代器是指用 iter (obj) 函数返回的对象
  • 迭代器是指用 next (it) 获取可迭代对象的数据

# iter & next

  • iter (iterable) 是指从可迭代对象中返回一个迭代器
  • next (iterator) 是指从迭代器中返回下一条记录，如果无法获取下一条记录，则则触发异常。因此迭代器只能只能往前取值，不能后退。

# 生成器

Python 中的生成器是最有用的特性之一，也是使用最不广泛的特性之一。生成器自动实现了迭代器协议，并对延迟操作提供了支持。所谓延迟操作是指在需要的时候产生结果，而不是立即产生结果。

# 生成器函数

常规函数定义，但是使用 yield 语句返回而不是 return 语句返回。yield 语句一次返回一个结果，在每个结果中间，挂起函数的状态，以便下次从它离开的地方继续执行。
