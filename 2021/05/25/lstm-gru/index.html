# LSTM

LSTM 结构是 RNN 结构的一种变体，其应用范围很广泛。LSTM 结构包含三种门，分别是输入门、遗忘门和输出门，三种门协同控制信息流的传递，从而一定程度上缓解长距离依赖问题。其主要结构可以通过以下 6 个公式进行表述：

# GRU

GRU 作为一种 LSTM 的变体，将忘记门和输入门合成一个单一的更新门，并重新设计了一个重置门，双门结构控制信息流传递，结构复杂度优于 LSTM，速度快，数据量较少时也可以超过 LSTM 的结果。
