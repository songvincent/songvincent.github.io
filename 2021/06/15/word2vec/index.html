# word2vec 之我见

word2vec 是 Word Embedding 的一种方法，由 2013 年提出。这种低维度的语义表征向量改善了 one-hot 编码的缺点 (矩阵稀疏 / 维度爆炸与语义缺失)，使得更多的机器学习 / 深度学习模型可以应用到 NLP 领域，对该领域具有里程碑的意义。

# 原理

word2vec 是一个简单的神经网络模型，由以下几层组成：

  • 1 个输入层，输入使用的是 one-hot 编码。
  • 1 个隐藏层，隐藏层的神经单元数量代表每一个词向量表示的维度大小，对于输入层与隐藏层之间的权值矩阵的形状是 [vocab_size,hidden_size]
  • 1 个输出层，输出层为 [vocab_size] 的大小的向量，值代表输出该词的概率。

# 模型

  • skip_gram 给定中心词预测上下文的词语 (模型结构 1-->1-->n)
  • CBOW 利用上下文词预测中心词 (模型结构 n-->1-->1)

# CBOW

分为两种情况，输入上下文为一个词时，被称为 simple CBOW Model；其上下文包含多个词时，被称为 CBOW Multi-Word Model。

  • simple CBOW Model (输入一个词，输出也为一个词)

    • input_layer 输入的 X 是单词的 one-hot 编码
    • 输入层到隐藏层之间有一个权重矩阵 W
    • 隐藏层到输出层也有一个权重矩阵 W'
    • 最终的输出要经过 softmax 函数，将输出向量中的每一个元素归一化到 0-1 之间的概率值，概率最大的就是预测的值

  • CBOW Multi-Word Context Model

    • 结构同上，只是输入由 1 个词变成了 C 个词，每个输入都会经过相同的权重 W，隐藏层 h 的值变为了多个词乘权重矩阵加和后取平均值。

# skip_gram

  • skip_gram 输入层到隐藏层的结构与 CBOW 相同，不同的是隐藏层到输出层，损失函数变成 C 个词的损失函数求和，隐藏层到输出层之间的矩阵也是共享的。
  • skip_gram 训练方法速度慢于 CBOW，但结果优于 CBOW

# 优化方法

Word2Vec 提出了两种加快训练速度的方法：

  • 负采样策略即通过重构训练集格式将原先的 N(vocan_size) 分类模型更改为多个二分类模型，提升速度
  • 层次 softmax 即通过构建 huffman 树将 N(vocan_size) 级别的 softmax 更改为 logN 级别的 sigmoid

# 局限性

  • word2vec 仅仅考虑了上下文信息，忽略了全局信息；
  • word2vec 仅仅考虑了上下文的共现性，忽略了彼此之间的顺序性
