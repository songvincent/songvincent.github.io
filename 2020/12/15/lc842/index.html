# leetcode842 将数组拆分成斐波那契序列

# 题目描述

具体题目信息可以通过链接跳转，作者将简单阐述题目要求。给定字符串 S ，将其分成斐波那契数列的形式，例如 S="123456789", 可分为 [123,456,789] (详细信息可以到题目中寻找)

# 解题方法

回溯：以前总是认为回溯就是递归，递归就很简单，没必要花费太多时间学习。开始写 leetcode 后，就感觉自己太年轻了，回溯的应用很广泛，能解决很多复杂问题。

本题即是如此，利用回溯递归地拆分原字符串的内容，从而解决问题，其要求主要有下面三点。

  • F.length >= 3
  • 每个块的数字一定不要以零开头，除非这个块是数字 0 本身。
  • 0 <= F[i] <= 2^31 - 1

# 学习与思考

这道题用到了回溯的思想，目前来说我并没有使用回溯解出任何一道题，感觉自己在这一方面还是有欠缺，只能是以后多刷题来培养自己的做题感觉。

回溯即递归，要注意的是结束条件，以本题为例，本题的结束条件为 (3+1) 个

  • 正常结束 (返回 true)

    • backtrack (i) 不断递归一直满足条件，直到返回 len(ans) >= 3

  • 异常结束 (返回 false)

    • 双 0 拼接式返回
    • 不断迭代导致生成的数大于最大值
    • 不断迭代导致生成的数大于答案列表的最后两数的和

还是要多写题啊，这种回溯题感觉不难，但就是写不出来。
